Excel Styling Functions

header_text_look

Apply a specific header style to a cell.

  • Parameters:
    • worksheet1: The worksheet object where the styling will be applied.
    • arg1: The cell address (e.g., 'A1') as a string to apply the style to.
  • Example:
    • import xlwings as xw

wb = xw.Book()
ws = wb.sheets['Sheet1']
header_text_look(ws, 'A1')

adaptive_header1_style

Apply an adaptive header style to a cell based on row and column indices.

  • Parameters:
    • worksheet1: The worksheet object where the styling will be applied.
    • arg1: A tuple containing the row and column indices (e.g., (1, 1)).
  • Example:
    • import xlwings as xw

wb = xw.Book()
ws = wb.sheets['Sheet1']
adaptive_header1_style(ws, (1, 1))
